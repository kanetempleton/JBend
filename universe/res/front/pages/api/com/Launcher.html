Constructors:

Constructor Description

Launcher()

default

  Initializes launcher with a maximum capacity of 10 threads and initializes the Console thread.

  Same as default but enables specification of maximum thread capacity.

Depreciated on initial release.

 Old way of initializing the launcher. Automatically loads and initiates all backend threads in a certain order.

Return type Method Description

loadThread(Runnable r)

 Adds any runnable thread object into the loading queue. Thread will begin running once the previous thread in the queue has been started.
void

startThreads()

 Begins launching loaded threads stored in the loading queue one at a time in the order which they were inserted.
